This example shows how to send streaming ACN (sACN) from a browser client using node.js and the node.js sacn library available at this link.

The sliders and number boxes below all send a DMX level (constrained to 0-255) to the node server whenever you change them. The node server then sends the sACN data to the listener. In other words, this won't work from a generic web server, you'll need to use the actual node.js script that's accompanying this file as your server in order to send sACN. You'll need that server to be on a computer on the same physical network as your sACN receiver as well.
